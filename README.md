Predictive Analytics Using Historical Data

Overview

This project demonstrates predictive analytics using historical sales data and Linear Regression. The model analyzes past sales trends and forecasts future sales values, helping businesses make data-driven decisions and plan for future growth.

Objectives

- Analyze historical sales data
- Build a predictive model using Linear Regression
- Forecast future sales trends
- Evaluate model accuracy
- Visualize historical and predicted sales data

Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn
- OpenPyXL

Dataset

The dataset contains 120 months of historical sales data used for training and testing the prediction model.

Features

- Data Loading and Preprocessing
- Linear Regression Model Training
- Future Sales Forecasting
- Accuracy Evaluation (R² Score)
- Data Visualization using Matplotlib
- Historical vs Predicted Sales Analysis

Project Workflow

1. Load historical sales data
2. Prepare independent and dependent variables
3. Train the Linear Regression model
4. Predict future sales values
5. Evaluate model performance
6. Visualize results using graphs

Results

- Successfully predicted future sales trends
- Achieved high model accuracy
- Generated clear visualizations for business insights

Learning Outcomes

- Predictive Analytics
- Regression Modeling
- Data Visualization
- Forecasting Techniques
- Machine Learning Fundamentals
